Documentação da AGSOFTY

Bem-vindo à documentação da AGSOFTY. Aqui você encontrará informações sobre como utilizar nossos serviços.

1. Introdução

Este guia fornece instruções detalhadas sobre os produtos e serviços da AGSOFTY, ajudando desenvolvedores e clientes a entenderem melhor sua funcionalidade.

2. Instalação

Para utilizar nossos serviços, siga os passos abaixo:

npm install agsofty-sdk

3. Configuração

Para configurar corretamente, edite o arquivo de configuração conforme abaixo:

{ "apiKey": "SUA_CHAVE_AQUI", "environment": "production" }
